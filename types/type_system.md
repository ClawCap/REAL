# REAL 人格类型体系

> REAL = Reading Everyone's Actual Life
> SBTI 问你"你觉得你是谁"，REAL 看你的数据说你是谁。

---

## 分类维度（5 维度，每维度 2 极）

从社交数据中提取 5 个核心维度，每个维度两极，组合成类型代码：

### 维度 1：内容倾向 — 创（C）vs 吃（E）
- **C = Create**：原创内容 > 转发/消费内容。你在输出。
- **E = Eat**：收藏/点赞/转发 > 原创。你在吃别人的内容。
- **判定依据**：原创帖数 / 总互动数（收藏+点赞+转发）

### 维度 2：社交温度 — 热（H）vs 冷（L）
- **H = Hot**：频繁互动，回复评论，跟人对线。社交温度高。
- **L = Low**：几乎不互动，发了就走，评论区空无一人。
- **判定依据**：评论回复率、互动频率、@别人次数

### 维度 3：兴趣广度 — 杂（W）vs 专（D）
- **W = Wide**：什么都看，什么都收藏，兴趣标签 > 5 个品类。
- **D = Deep**：只关注 1-2 个领域，深度钻研型。
- **判定依据**：内容标签/主题的分散度（熵值）

### 维度 4：表里一致 — 真（R）vs 演（P）
- **R = Real**：发的内容和收藏/点赞的内容高度一致。表里如一。
- **P = Play**：发的是一套，收藏的是另一套。你在演。
- **判定依据**：公开内容主题 vs 私密行为（收藏/点赞）主题的重叠度

### 维度 5：品味层 — 小众（U）vs 大众（M）
- **U = Underground**：关注小众博主/冷门内容/独立创作者。品味独特。
- **M = Mainstream**：关注百万粉大V/热门话题/爆款内容。跟着大众走。
- **判定依据**：关注列表中的博主粉丝数中位数、收藏内容的热度分布

## 类型描述风格

每个类型的描述包含：
1. **一句话定义**（标签感）
2. **数据证据**（"你的收藏比发帖多 47 倍"）
3. **扎心解读**（好笑的、自嘲型的）
4. **隐藏优势**（每个类型都有正面解读）
5. **代表行为**（"你一定做过这些事"）

### 描述示例：ELWRM「囤货仓鼠」🐹

```
你是 ELWRM —— 囤货仓鼠 🐹

数据说：
📊 原创内容：12 条 | 收藏：2,341 条 | 点赞：892 条
💬 互动率：< 1%（评论区从不出现你的身影）
🏷️ 收藏横跨 8 个品类（美食/旅行/穿搭/理财/健身/养猫/装修/摄影）
🎯 关注列表中位数粉丝：120万（你追的都是大V和头部博主）

扎心解读：
你是互联网的松鼠——默默囤坚果，但从不吃。
收藏夹里有 2341 条内容，大概打开过 3 条。
你追的都是百万粉的大博主，算法推什么你收什么。
不是品味不好，是品味太大众了——但你不在乎，好用就行。

隐藏优势：
你是最高效的信息筛选器。不花时间社交，不花时间创作，
所有精力都用来"看"和"存"。你的收藏夹就是你的第二大脑。
在别人还在刷的时候，你已经收藏完回去睡觉了。

你一定做过这些事：
✅ 收藏夹分了十几个文件夹，每个里面都是灰尘
✅ 关注了一堆百万粉博主，但从没给谁评论过
✅ 半夜刷到"必收藏"帖子，一键收藏，第二天忘了
✅ 有人推荐小众博主，你看了看粉丝数，默默划走
```

## 分类算法逻辑

### 维度 1：C vs E
```
原创占比 = 原创帖数 / (原创帖数 + 转发数 + 收藏数 + 点赞数)
> 0.3 → C（你在创造）
≤ 0.3 → E（你在消费）
```

### 维度 2：H vs L
```
互动率 = (回复评论数 + @提及数 + 互动帖数) / 总帖数
> 0.2 → H（社交热）
≤ 0.2 → L（社交冷）
备选：评论区回复率、日均互动次数
```

### 维度 3：W vs D
```
兴趣熵值 = -Σ(p_i × log(p_i))，其中 p_i 为各主题占比
> 1.5 → W（兴趣广）
≤ 1.5 → D（兴趣深）
备选：TOP 1 主题占比 > 50% → D，否则 W
```

### 维度 4：R vs P
```
表里一致度 = 公开内容标签 ∩ 私密行为标签 / 公开内容标签 ∪ 私密行为标签
> 0.5 → R（真实）
≤ 0.5 → P（在演）
注意：如果只有公开数据（没有收藏/点赞），该维度标注"数据不足"并默认为 R
```

### 维度 5：U vs M
```
关注列表粉丝中位数 = median(关注的所有博主的粉丝数)
< 10万 → U（小众品味）
≥ 10万 → M（大众品味）

辅助判定：
- 收藏内容的平均互动量（点赞/转发数）：< 1000 → 倾向 U
- 关注列表中 < 1万粉博主的占比 > 50% → 强 U
- 如果关注列表无法获取，用收藏内容的热度分布代替
```

### 匹配度计算
每个维度不是非黑即白——计算距离阈值的偏离程度：
- 偏离越大 → 匹配度越高（"你是一个非常典型的 ELWRM"）
- 接近阈值 → 匹配度较低（"你在 U 和 M 之间摇摆"）

最终匹配度 = 5 个维度匹配度的加权平均（0-100%）
